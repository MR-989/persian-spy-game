# دليل النشر من الجوال — Persian Spy Game (خدمة واحدة)

السيرفر يبني الواجهة ويقدّمها، ويشغّل Socket.IO على نفس الرابط.
نشر واحد فقط، رابط واحد، بدون إعداد CORS أو متغيّرات بيئة.

تحتاج: حساب GitHub + حساب Render (الاثنان مجانيان ومن المتصفح).

---------------------------------------------------------------
## الخطوة 1 — ارفع الكود على GitHub (من الجوال)
1. فُكّ ضغط ملف `persian-spy-game-production.zip`.
2. افتح github.com وسجّل دخول. اضغط (+) ثم New repository.
3. سمِّه مثلاً `persian-spy-game`، اجعله Public، أنشئه.
4. في صفحة المستودع: زر "Add file" ثم "Upload files".
5. ارفع كل محتويات المجلد `persian-spy-game` (مجلد server و client و scripts
   وملفات package.json و render.yaml ... إلخ). لا ترفع مجلد node_modules.
6. اضغط Commit changes.

> ملاحظة: رفع المجلدات من متصفح الجوال أحيانًا يكون متعِبًا. الأسهل لو أمكن:
> استخدم تطبيق GitHub، أو ارفع من تابلت، أو من أي كمبيوتر متاح لمرة واحدة فقط.

---------------------------------------------------------------
## الخطوة 2 — انشر على Render
طريقة أ (الأسهل — Blueprint يقرأ render.yaml):
1. افتح render.com وسجّل الدخول عبر GitHub.
2. New ثم Blueprint. اختر مستودع `persian-spy-game`.
3. Render يقرأ `render.yaml` ويُنشئ خدمة واحدة باسم persian-spy-game.
4. اضغط Apply / Create. انتظر البناء (٢ إلى ٥ دقائق).

طريقة ب (يدوي بدون blueprint):
1. New ثم Web Service. اختر المستودع.
2. Runtime: Node. Build Command: `npm run build`. Start Command: `npm start`.
3. Health Check Path: `/health`. Plan: Free. اضغط Create.

---------------------------------------------------------------
## الخطوة 3 — العب
- بعد ما تصير الحالة Live، نسخ الرابط مثل:
  `https://persian-spy-game.onrender.com`
- افتح الرابط من جوالك: اكتب اسمك ثم «ساخت اتاق» وخذ كد الاتاق.
- افتح نفس الرابط على جوال صديقك (أو تبويب جديد) وادخل بالكود.
- تحتاج ٣ لاعبين على الأقل: شروع بازی ثم «دیدم» من الجميع ثم التصويت ثم النتيجة.

---------------------------------------------------------------
## تحقّق سريع
- افتح `https://<your-app>.onrender.com/health`
  لازم يرجّع: {"ok":true,"service":"persian-spy-game"}

## ملاحظات
- الخطة المجانية في Render "تنام" بعد فترة خمول، فأول فتح بعد النوم قد
  يأخذ ٣٠ إلى ٦٠ ثانية ليصحى. بعدها يصير سريع.
- حالة اللعبة في الذاكرة: إعادة تشغيل الخدمة تمسح الغرف الحالية (طبيعي للعبة سريعة).
